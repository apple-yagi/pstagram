# Pstagram

node v12.15.0  
vue 2.6.10  
vue-cli 3.0.1

## Project setup

このコマンドを打つと Vue project を setup できます。

```
git clone https://github.com/apple-yagi/pstagram.git
cd pstagram
npm install
```

### Compiles and hot-reloads for development

このコマンドでプロジェクトを実行できます。

```
npm run serve
```

### Web ページの確認

http://localhost:8080 で確認できます
